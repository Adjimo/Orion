# 🌟 Orion

> Le chasseur — webapp PWA pour gamifier ta vie de **prépa MP\*** + **trail/course**.

---

## 🎮 Features

- **Splash screen** "ORION" stylé (constellation + titre doré)
- **Navigation tabbar** : Accueil / Quêtes / Sport / Stats / Profil
- **Système XP** avec niveaux et 9 rangs (Mortel → ... → Orion)
- **7 compétences RPG** : Endurance, Montagne, Maths, Physique, SI, Lettres, Discipline
- **Saisie de travail unifiée par matière** avec recalcul automatique de l'XP
- **Quêtes hebdomadaires** : 3 sorties course (auto), 30h travail (auto), 3h langues (auto), 7 nuits 7h+ (manuel)
- **Import GPX** + carte Leaflet, calcul automatique de la distance, D+, allure, splits 5K/10K
- **Records personnels** sport + travail (plus longue sortie, plus grosse semaine, etc.)
- **Compte à rebours concours MP\*** (X/ENS, Mines-Ponts, Centrale, CCINP) avec **% année écoulée**
- **Export/Import JSON** pour sauvegarder ta progression
- **PWA installable** sur Android/iOS, fonctionne hors-ligne

---

## 🧹 Réinitialiser ta progression

Une fois l'app installée :
1. Onglet **Profil** → **Zone dangereuse**
2. Clique sur **🗑️ Tout effacer** → confirme deux fois

Tu repars à niveau 1, 0 XP, rien.

---

## 💾 Sauvegarder tes données

Tes données vivent **uniquement** dans le navigateur. Si tu vides Chrome ou changes de téléphone : tout est perdu.

Pour ne rien perdre :
- **Profil** → **Sauvegarde** → **📥 Exporter (JSON)**
- Le fichier `orion-backup-YYYY-MM-DD.json` se télécharge
- Mets-le sur Google Drive / par email à toi-même
- Pour restaurer : **📤 Importer une sauvegarde**
